## Tools Covered

- **ELAN** - Multimedia annotation tool
- **WebAnno** - Web-based annotation platform
- **CATMA** - Computer Assisted Text Markup and Analysis
- **AntConc** - Corpus analysis toolkit
- **R packages** - quanteda, tidytext, and others
